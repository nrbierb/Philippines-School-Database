WebSchoolsDatabase by nrbierb

Web based database system for school systems including students, teachers, schools, and upper level organizations

A fully web based database for school systems including students, teachers, schools, and upper level organizations. The Google App-Server database system is used for all database and web support. All server code is in Python. The Django web framework provides form generation and processing. There is extensive javascrpt and Ajax in the user pages with Jquery as the primary library.

Dependencies

Google App-Engine Django Web Framework jQuery jQuery-UI

Install

Create google appengine account and upload full source. Contact me for further info
